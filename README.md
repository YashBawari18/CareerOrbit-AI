# 🚀 CareerOrbit AI

**CareerOrbit AI** is a cutting-edge platform designed to help professionals navigate the rapidly evolving job market. By leveraging AI-driven analytics, 3D visualizations, and predictive modeling, we turn career uncertainty into a calculated path for growth.

---

## ✨ Key Features

- **3D Skill Orbit**: Interactive visualization of your skills in a neural-like cluster, showing how they connect to market demands.
- **Predictive Career Analytics**: Advanced models that simulate market relevance and predict your next best career pivot.
- **Skill Decay Tracking**: Automated tracking of technical skill half-life to identify when it's time to upskill.
- **Non-Linear Career Roadmaps**: Tailored paths that account for jumps across sectors, roles, and geographies.
- **Algorithmic Fairness**: Built-in fairness metrics to ensure career opportunities are equitable across all demographics.
- **Premium User Experience**: Immersive design featuring glassmorphism, cinematic animations with Framer Motion and a refined light-mode aesthetic.

---

## 🛠️ Tech Stack

### Frontend
- **Framework**: [React](https://reactjs.org/) (Vite)
- **Styling**: Vanilla CSS with Modern Glassmorphism
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Visualizations**: [Three.js](https://threejs.org/), [React Three Fiber](https://r3f.docs.pmnd.rs/), [Recharts](https://recharts.org/)
- **Routing**: [React Router DOM](https://reactrouter.com/)

### Backend
- **Runtime**: [Node.js](https://nodejs.org/)
- **Framework**: [Express.js](https://expressjs.com/)
- **Database**: [PostgreSQL](https://www.postgresql.org/) (Relational) & [MongoDB](https://www.mongodb.com/) (Document)
- **ORM/ODM**: [Sequelize](https://sequelize.org/) & [Mongoose](https://mongoosejs.com/)
- **Authentication**: JWT with BcryptJS

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v16+)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)
- [PostgreSQL](https://www.postgresql.org/) / [MongoDB](https://www.mongodb.com/) (Local or Cloud)

### 1. Clone the Repository
```bash
git clone https://github.com/YashBawari18/CareerOrbit-AI.git
cd CareerOrbit-AI
```

### 2. Setup the Backend
```bash
cd server
npm install
# Configure your .env file with DATABASE_URL, JWT_SECRET, etc.
npm run dev
```

### 3. Setup the Frontend
```bash
# From the root directory
cd client
npm install
npm run dev
```
